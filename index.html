Quien soy

Soy un desarrollador frontend y en la actualidad me enfoco en la elaboración de páginas web. En mi primera etapa en informática, participe en varios proyectos, por ejemplo, un sistema de ingresos de datos automatizado de una biblioteca estudiantil, entre otros.

Unos años más tarde retomo mis estudios añadiendo nuevas tecnologías como: Bootstrap, Sass, Git y React.

HTML5

HTML 5 es un lenguaje de marcado de hipertexto que se utiliza para estructurar y desplegar una página web y sus contenidos. En el podras maquetar con una serie de elementos que te permitiran agregar imagenes, videos, parrafos, encabezados, listas, tablas etc.

CSS3

CCS es un leguaje de estilos en cascada. Con esta herramienta podemos definir la aparencia de nuestras paginas web de la manera que se nos ocurra. agregar estilos, cambiar colores, tamaños etc.

JAVASCRIPT

JavaScript es un lenguaje de programación que se utiliza para crear páginas web interactivas. Es interpretado por los navegadores web y se utiliza para crear páginas dinámicas. Es un lenguaje de alto nively es considerado uno de los lenguajes mas utilizdos en la actualidad.

BOOTSTRAP 5

Bootstrap, es un framework originalmente creado por Twitter, que permite crear interfaces web con CSS y JavaScript, cuya particularidad es la de adaptar la interfaz del sitio web al tamaño del dispositivo en que se visualice.

SASS

Es un preprocesador CSS que nos permite generar, de manera automática, hojas de estilo, añadiéndoles características que no tiene CSS hasta ahora. como pueden ser variables, funciones, selectores anidados etc.

REACT

React es una biblioteca javascript diseñada para crear interfaces de usuario interactivas de forma sencilla. acelera el tiempo de desarrollo haciendolo ahora mismo uno de los frameworks mas utilizados por la comunidad.
